# Deltax

## Built With

- Django
- Sqlite3
- JavaScript
- HTML
- CSS
- Python

## Installation 

- `python3 -m venv yourenvname`
- `activate yourenvname`
- `git clone https://github.com/lavishkhandelwal/deltax.git`
- `pip install -r requirements.txt`
- `cd deltax`
- `python manage.py runserver`
